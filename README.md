# CoffeeMore Website

CoffeeMore iOS アプリのウェブサイト（利用規約・プライバシーポリシー）です。

## 公開URL

GitHub Pages で公開されています。

- サイト: https://coffeemore.net/
- 利用規約: https://coffeemore.net/terms/
- プライバシーポリシー: https://coffeemore.net/privacy/

各ページは日本語・英語・韓国語・中国語に対応しています。
