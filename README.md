# CoffeeBook Website

CoffeeBook iOS アプリのウェブサイト（利用規約・プライバシーポリシー）です。

## 公開URL

GitHub Pages で公開されています。

- サイト: https://yy333512.github.io/coffeebook-website/
- 利用規約: https://yy333512.github.io/coffeebook-website/terms/
- プライバシーポリシー: https://yy333512.github.io/coffeebook-website/privacy/

各ページは日本語・英語・韓国語・中国語に対応しています。

## ホスティング

- **配信**: GitHub Pages（HTTPS 強制）
- **配信元**: `main` ブランチのルート (`/`)
- **ビルド**: 静的サイト（Jekyll 無効化のため `.nojekyll` を配置）

`main` ブランチに push すると自動で公開サイトに反映されます。

## 構成

```
.
├── index.html      # トップページ
├── style.css       # スタイル
├── privacy/        # プライバシーポリシー（多言語）
└── terms/          # 利用規約（多言語）
```
